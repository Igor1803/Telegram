# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
pip install -r requirements.txt
```

## Шаг 2: Настройка переменных окружения

1. Скопируйте `.env.example` в `.env`:
```bash
copy .env.example .env
```

2. Откройте `.env` и заполните:
```
TELEGRAM_BOT_TOKEN=ваш_токен_бота
OPENAI_API_KEY=ваш_ключ_openai
```

### Как получить токен Telegram-бота:

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен в `.env`

### Как получить API ключ OpenAI:

1. Зарегистрируйтесь на [OpenAI Platform](https://platform.openai.com/)
2. Перейдите в раздел [API Keys](https://platform.openai.com/api-keys)
3. Создайте новый ключ
4. Скопируйте ключ в `.env`

## Шаг 3: Запуск бота

```bash
python bot.py
```

Вы должны увидеть сообщение:
```
INFO - Бот запущен...
```

## Шаг 4: Тестирование

1. Откройте Telegram
2. Найдите вашего бота по имени, которое вы указали при создании
3. Отправьте команду `/start`
4. Начните диалог с ботом

## Проверка работы

Бот должен:
- ✅ Ответить приветственным сообщением
- ✅ Задавать естественные вопросы
- ✅ Вести диалог без шаблонов
- ✅ Завершить диалог и сформировать саммари

## Команды бота

- `/start` - начать новый диалог
- `/reset` - сбросить текущий диалог

## Устранение проблем

### Ошибка "TELEGRAM_BOT_TOKEN не установлен"
- Проверьте, что файл `.env` существует
- Проверьте, что токен указан правильно (без кавычек)

### Ошибка "OPENAI_API_KEY не установлен"
- Проверьте, что ключ указан в `.env`
- Убедитесь, что у вас есть доступ к OpenAI API

### Бот не отвечает
- Проверьте, что бот запущен (должно быть сообщение "Бот запущен...")
- Проверьте интернет-соединение
- Проверьте логи на наличие ошибок

### Ошибки при работе с OpenAI API
- Проверьте баланс на аккаунте OpenAI
- Убедитесь, что используете правильную модель (gpt-4o)
- Проверьте лимиты API
